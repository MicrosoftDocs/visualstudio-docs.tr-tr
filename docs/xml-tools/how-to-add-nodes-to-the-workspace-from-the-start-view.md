# <a name="how-to-add-nodes-to-the-workspace-from-the-start-view"></a>Nasıl yapılır: düğümleri Başlat görünümünden çalışma alanına ekleyin

Bu konu genel düğümlerine eklemeyi açıklar [XML şema Tasarımcısı çalışma](../xml-tools/xml-schema-designer-workspace.md) gelen [Başlangıç'ı](../xml-tools/start-view.md).

## <a name="to-add-nodes-to-the-workspace-from-the-start-view"></a>Düğüm başlangıç görünümünden çalışma alanına eklemek için

1.  Adımları [nasıl yapılır: oluşturma ve bir XSD şema dosyası düzenleme](../xml-tools/how-to-create-and-edit-an-xsd-schema-file.md).

2.  XML Düzenleyicisi'nde sağ tıklatıp **Görünüm Tasarımcısı**.

     Başlangıç görünümü görüntülenir.

     İçinde **şema ayarlamak ayrıntılarını** bölümünde, iki olduğuna dikkat edin (**2**) **genel karmaşık türler**.

3.  Tıklayın **ekleme** sağındaki bağlantı **genel karmaşık türler** başlığı.

     `USAddress` Ve `PurchaseOrderType` karmaşık tür düğüm, grafik görünümü tasarım yüzeyine görünür.
