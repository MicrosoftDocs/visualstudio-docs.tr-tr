# <a name="command-line-profiling-tools-reference"></a>Komut satırı profil oluşturma araçları başvurusu
Komut satırı performans araçları uygulamalarını komut isteminden profil programcıların izin verin. Aşağıdaki komut satırı profil oluşturma araçları, komut isteminden profil kodu izin verir.  
  
## <a name="in-this-section"></a>Bu bölümde  
 [VSInstr](../profiling/vsinstr.md)  
 VSInstr.exe izleme seçeneklerini listeler.  
  
 [VSInstr Uyarıları](../profiling/vsinstr-warnings.md)  
 Vsınstr aracı tarafından verilen uyarılar listeler.  
  
 [VSPerfCLREnv](../profiling/vsperfclrenv.md)  
 Yönetilen kod profil oluşturma için ortamı yapılandırmak için VSPerfCLREnv.exe seçeneklerini listeler.  
  
 [VSPerfCmd](../profiling/vsperfcmd.md)  
 Performans verilerini toplamak için VSPerfCmd.exe seçeneklerini listeler.  
  
 [VSPerfMon](../profiling/vsperfmon.md)  
 Veri toplama toplama VSPerfMon.exe seçeneklerini listeler.  
  
 [VSPerfReport](../profiling/vsperfreport.md)  
 Raporları oluşturmak için komut satırı seçenekleri VSPerfReport.exe listeler.
