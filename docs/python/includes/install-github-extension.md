VS 'nin içinden GitHub depolarıyla çalışmak için, Visual Studio için GitHub uzantısını yüklemeniz gerekir. Bunu yapmak için, Visual Studio yükleyicisi 'ni çalıştırın, **Değiştir**' i seçin ve **ayrı bileşenler** sekmesini seçin. **Kod araçları** bölümünde aşağı kaydırın, **Visual Studio için GitHub Uzantısı**' nı seçin ve **Değiştir**' i seçin.
