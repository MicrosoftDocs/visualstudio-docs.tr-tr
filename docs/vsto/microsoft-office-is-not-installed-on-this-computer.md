# <a name="microsoft-office-is-not-installed-on-the-computer"></a>Microsoft Office bilgisayarda yüklü değil
  Visual Studio proje türüne karşılık gelen Microsoft Office uygulaması bulunamıyor. Bir VSTO eklenti projesi geliştiriyorsanız, hata ayıklama veya projeyi çalıştırın, anlamına gelir. Belge düzeyi projesi oluşturmak çalışıyorsanız, bu proje oluşturulamıyor anlamına gelir.  
  
## <a name="to-correct-the-error"></a>Hatayı düzeltmek için  
  
1.  Birincil birlikte çalışma derlemeleri de dahil olmak üzere proje türü ile ilişkili Microsoft Office uygulamasını yükleyin. Microsoft Office'in desteklenen sürümleri hakkında daha fazla bilgi için bkz: [Office çözümleri geliştirmek için bilgisayarı yapılandırma](../vsto/configuring-a-computer-to-develop-office-solutions.md).
