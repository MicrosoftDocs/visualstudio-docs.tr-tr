# <a name="metadata_address_param"></a>METADATA_ADDRESS_PARAM
Bu yapı bir yöntemin veya işlevin parametresini temsil eder.

## <a name="syntax"></a>Syntax

```cpp
typedef struct _tagMETADATA_ADDRESS_PARAM {
   _mdToken tokMethod;
   _mdToken tokParam;
   DWORD    dwIndex;
} METADATA_ADDRESS_PARAM;
```

```csharp
public struct METADATA_ADDRESS_PARAM {
   public int  tokMethod;
   public int  tokParam;
   public uint dwIndex;
}
```

## <a name="members"></a>Üyeler
 `tokMethod`\
 Parametresinin bir parçası olduğu yöntemin KIMLIĞI.

 `tokParam`\
 Parametrenin KIMLIĞI.

 `dwIndex`\
 Parametre listesindeki parametresinin dizini.

## <a name="remarks"></a>Açıklamalar
 Bu yapı, [DEBUG_ADDRESS_UNION](../../../extensibility/debugger/reference/debug-address-union.md) `dwKind` `DEBUG_ADDRESS_UNION` yapı alanı `ADDRESS_KIND_PARAM` ( [ADDRESS_KIND](../../../extensibility/debugger/reference/address-kind.md) numaralandırmasından bir değer) olarak ayarlandığında DEBUG_ADDRESS_UNION yapısındaki birleşimin bir parçasıdır.
