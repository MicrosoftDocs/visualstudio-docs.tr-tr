# <a name="stepunit"></a>STEPUNIT
Step unit atlamak için belirtir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>Üyeler  
 STEP_STATEMENT  
 Deyimi tarafından adımları.  
  
 STEP_LINE  
 Adımları çizgi.  
  
 STEP_INSTRUCTION  
 Yönerge adımlarla.
