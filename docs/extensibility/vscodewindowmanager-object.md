[eskı API 'Yi kullanarak kod pencerelerini özelleştirme](/previous-versions/visualstudio/visual-studio-2015/extensibility/customizing-code-windows-by-using-the-legacy-api?preserve-view=true&view=vs-2015).

|Arabirim|Açıklama|
|---------------|-----------------|
