[eskı API 'Yi kullanarak kod pencerelerini özelleştirme](../vs-2015/extensibility/customizing-code-windows-by-using-the-legacy-api.md?view=vs-2015&preserve-view=true).

|Arabirim|Açıklama|
|---------------|-----------------|
