## <a name="syntax"></a>Sözdizimi

```C++
HRESULT get_lengthProlog ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>Parametreler
 `pRetVal`

dışı Prolog kodunun bayt sayısını döndürür.
