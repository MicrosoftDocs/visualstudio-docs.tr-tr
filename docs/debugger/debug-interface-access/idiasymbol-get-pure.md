```C++
HRESULT get_pure ( 
   BOOL* pRetVal
);
```

#### <a name="parameters"></a>Parametreler

dışı `TRUE` İşlevin saf sanal olup olmadığını döndürür; Aksi takdirde, döndürür `FALSE` .
