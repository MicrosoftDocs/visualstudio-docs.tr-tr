## <a name="syntax"></a>Söz dizimi

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>Parametreler
 pRetVal

dışı Eklenen kaynakların sayısını döndürür.
