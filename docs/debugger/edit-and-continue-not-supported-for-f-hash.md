# <a name="edit-and-continue-not-supported-for-f"></a>Düzenle ve Devam Et F#'de Desteklenmez #
Düzenle ve devam et F # kodunda hata ayıklama yaparken desteklenmez. F # kodunda yapılan düzenlemeler, hata ayıklama oturumu sırasında olası ancak kaçınılmalıdır. Kod değişiklikleri hata ayıklama oturumu sırasında uygulanmaz. Bu nedenle, F # kodu, hata ayıklarken yapılan düzenlemeler ayıklanacak kodla eşleşmiyor kaynak kodunda neden olur.
