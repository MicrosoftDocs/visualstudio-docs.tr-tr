# <a name="supplying-xml-code-comments"></a>XML Kodu Açıklamalarını Sağlama
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Kaynak kodu dosyalarındaki Yorumları IntelliSense olarak kullanılabilir hale getirebilirsiniz. Visual Basic, C# veya C++ derleyici komut satırına/doc anahtarını eklediğinizde, IntelliSense belgelerinin temeli görevi gören bir. xml dosyası oluşturulur.

 Visual Basic kodu belgeleme hakkında daha fazla bilgi için bkz. [nasıl yapılır: XML belgeleri oluşturma](https://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).

 C# dilinde kod belgeleme hakkında daha fazla bilgi için bkz. [XML belge açıklamaları](https://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).

 C++ ' ta kod belgeleme hakkında daha fazla bilgi için bkz. [Comments](https://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) ve [XML belgeleri](https://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
