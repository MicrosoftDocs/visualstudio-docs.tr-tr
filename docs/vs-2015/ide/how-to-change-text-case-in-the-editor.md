# <a name="how-to-change-text-case-in-the-editor"></a>Nasıl Yapılır: Düzenleyicide Büyük/Küçük Harf Değiştirme
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Seçilen metnin büyük/küçük harflere veya tüm küçük harflere dönüştürülmesini sağlamak için menü komutlarını veya klavye kısayollarını kullanabilirsiniz.

> [!NOTE]
> Gördüğünüz iletişim kutuları ve menü komutları, etkin ayarlarınıza veya sürümüne bağlı olarak **Yardım** bölümünde açıklananlardan farklı bir durum içerebilir. Ayarlarınızı değiştirmek için **Araçlar** menüsünden **Içeri ve dışarı aktarma ayarları** ' nı seçin. Daha fazla bilgi için bkz. [Visual Studio 'Da geliştirme ayarlarını özelleştirme](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="procedure"></a>Yordam

#### <a name="to-switch-text-to-upper-case"></a>Metni büyük harflere geçmek için

1. Dönüştürmek istediğiniz metni seçin.

2. **Düzenle** menüsünde **Gelişmiş**' i seçin.

3. Metni büyük harflere dönüştürmek için, **büyük harf yap**' ı seçin veya **CTRL** + **SHIFT** + **U**tuşlarına basın.

     veya

     Metni küçük harfe dönüştürmek için **küçük harf yap**' ı seçin veya **CTRL** + **U**tuşlarına basın.

    > [!TIP]
    > Bu değişiklikten önce önceki durum biçimlendirmesine dönmek için, **Düzenle** menüsünden **geri al** ' ı seçin.
