# <a name="advanced-security-settings-dialog-box"></a>Gelişmiş Güvenlik Ayarları İletişim Kutusu
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
Bu iletişim kutusu, ilgili bölgede hata ayıklama için güvenlik ayarları belirtmenize olanak sağlar.  
  
 Bu iletişim kutusuna erişmek için bir proje düğümü seçin **Çözüm Gezgini**ve ardından **proje** menüsünde tıklatın **özellikleri**. Zaman **Proje Tasarımcısı** görünen tıklayın **güvenlik** sekmesi. Üzerinde **güvenlik** sayfasında **ClickOnce güvenlik ayarlarını etkinleştirme**, tıklayın **kısmi güven uygulamasıdır**ve ardından **Gelişmiş**.  
  
## <a name="uielement-list"></a>UIElement Listesi  
 **Bu uygulama için seçili izin kümesi ile hata ayıklama**  
 Bu onay kutusunu seçerseniz, iznin seçili **güvenlik** sayfası, hata ayıklama sırasında kullanılır. Varsayılan olarak, bu seçenek seçilidir.  
  
 İş için bir güvenlik bölgesi hata ayıklama için bu seçeneği etkinleştirilmelidir; Ayrıca, **Visual Studio barındırma işlemini etkinleştir** seçeneği (kullanılabilir **hata ayıklama** sayfasının **Proje Tasarımcısı**) etkinleştirilmesi gerekir.  
  
 WPF Web tarayıcı uygulaması projeleri için **bu uygulama için seçili izin kümesi ile hata ayıklama** seçeneği işaretli ve devre dışı.  
  
 **Gulamaya erişim ver**  
 Bu onay kutusunu seçerseniz, uygulama Web sitesi veya sunucu paylaşımı üzerinde yayımlandıktan erişebilirsiniz. Varsayılan olarak, bu seçenek seçilidir.  
  
 **Bu uygulama aşağıdaki URL'den yüklendiyse hata ayıklayın**  
 Web sitesi veya karşılık gelen sunucu paylaşımı erişmek uygulamayı izni varsa **yükleme URL'si** , belirtilen **Yayımla** sayfasında, bu URL'yi buraya yazın. Bu seçenek yalnızca, **gulamaya erişim izni** seçilir.
