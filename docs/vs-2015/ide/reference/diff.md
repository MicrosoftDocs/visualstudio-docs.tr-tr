# <a name="diff"></a>/Diff
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

İki dosyayı karşılaştırır. Farklar özel bir Visual Studio penceresinde görüntülenir.

## <a name="syntax"></a>Söz dizimi

```
devenv /Diff SourceFile, TargetFile, [SourceDisplayName],[TargetDisplayName]
```

## <a name="arguments"></a>Bağımsız değişkenler
 `SourceFile` Gerekli. Karşılaştırılacak ilk dosyanın tam yolu ve adı.

 `TargetFile` Gerekli. Karşılaştırılacak ikinci dosyanın tam yolu ve adı

 `SourceDisplayName` Seçim. İlk dosyanın görünen adı.

 `TargetDisplayName` Seçim. İkinci dosyanın görünen adı.
