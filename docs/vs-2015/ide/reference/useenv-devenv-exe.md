# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)

## <a name="syntax"></a>Syntax

```
Devenv /useenv
```

 Aşağıdaki örnek, [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] ortam değişkenlerini **VC + + dizinleri** iletişim kutusuna başlatır ve yükler.

```
Devenv.exe /useenv
```
