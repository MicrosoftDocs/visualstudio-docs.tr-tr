# <a name="how-to-debug-inline-assembly-code"></a>Nasıl Yapılır: Satır İçi Derleme Kodunda Hata Ayıklama
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

NOTUN
> Gördüğünüz iletişim kutuları ve menü komutları, etkin ayarlarınıza veya sürümüne bağlı olarak yardım bölümünde açıklananlardan farklı bir durum içerebilir. Ayarlarınızı değiştirmek için Araçlar menüsünden Içeri ve dışarı aktarma ayarları ' nı seçin. Daha fazla bilgi için bkz. [Visual Studio 'Da geliştirme ayarlarını özelleştirme](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
 Hata ayıklayıcı, satır içi derleme kodu, **ayrıştırma** penceresi ve **Yazmaçları** penceresinde hata ayıklama için iki pencere sağlar.  
  
## <a name="procedure"></a>Yordam  
  
#### <a name="to-debug-inline-assembly-code"></a>Satır içi derleme kodunda hata ayıklamak için  
  
1. Bütünleştirilmiş kod yönergelerini görüntülemek için **ayrıştırma** penceresini kullanın.  
  
2. Kayıt içeriğini görüntülemek için **Yazmaçları** penceresini kullanın.
