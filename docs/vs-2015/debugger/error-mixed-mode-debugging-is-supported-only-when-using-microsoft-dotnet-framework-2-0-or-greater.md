# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Hata: Karışık modda hata ayıklama yalnızca Microsoft .NET Framework 2.0 veya daha yenisi kullanılırken desteklenir
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Karma yerel ve yönetilen kodda hata ayıklamak için [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] 2,0, 3,0 sürümüne sahip olmanız gerekir. 3,5 veya 4,0. Öğesinin önceki sürümleriyle karışık modda hata ayıklama [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] desteklenmez.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)]Sürümünü 2,0, 3,0, 3,5 veya 4,0 sürümüne yükseltin.
