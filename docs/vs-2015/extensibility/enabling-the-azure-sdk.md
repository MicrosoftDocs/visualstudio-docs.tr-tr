# <a name="enabling-the-azure-sdk"></a>Azure SDK'yı Etkinleştirme
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

.NET için Azure SDK, Azure 'da ölçeklenebilir Web uygulamaları ve Hizmetleri oluşturmanıza, yapılandırmanıza, oluşturmanıza, hata ayıklamanıza, çalıştırmanıza ve paketlemenize yardımcı olur.  
  
 Azure SDK 'sını **Yeni proje** iletişim kutusundan indirebilirsiniz.  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Azure SDK 'sını etkinleştirmek için  
  
1. **Dosya** menüsünde Yeni bir proje oluşturun. ( **Yeni** ' ye ve ardından **Proje**' ye tıklayın.)  
  
2. **Visual Basic** veya **C#** düğümünü genişletin ve ardından **bulut**' a tıklayın.  
  
3. Orta bölmede **.net Için Azure SDK 'Yı al**' ı seçin ve ardından **Tamam**' a tıklayın.  
  
     Yükleyicideki yönergeleri izleyin.  
  
    > [!NOTE]
    > Azure araçlarını yüklemek için Internet Information Services 7,0 ve ASP.NET uygulama geliştirme bileşenlerinin yüklü olması gerekir. Bu bileşenler yüklü değilse, Azure Araçları kurulumu sizden bunları yüklemenizi ister.
