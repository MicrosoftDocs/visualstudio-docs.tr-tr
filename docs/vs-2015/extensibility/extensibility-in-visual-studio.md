# <a name="extensibility-in-visual-studio"></a>Visual Studio'da Genişletilebilirlik
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio size çok sayıda genişletilebilirlik seçeneği sunar. Kendi SDK 'larınızı oluşturabilir, Windows, Windows Phone ve Azure SDK 'larını kullanabilirsiniz (Visual Studio 'nun bir parçası olarak yüklenir) ve Visual Studio 'nun kendisini genişletebilirsiniz.

## <a name="extend-visual-studio"></a>Visual Studio’yu Genişletme
 Visual Studio SDK 'sını kullanarak Visual Studio 'nun her bir bölümünü genişletebilirsiniz: komutlar, menüler, pencereler, düzenleyiciler ve projeler. Daha fazla bilgi edinmek için bkz. [Visual STUDIO SDK](../extensibility/visual-studio-sdk.md).

## <a name="create-your-own-sdks"></a>Kendi SDK 'Larınızı oluşturun
 Kendi platformunuzu ve uzantı SDK 'larını oluşturma, paketleme ve dağıtma hakkında bilgi edinin: [yazılım geliştirme seti oluşturma](../extensibility/creating-a-software-development-kit.md).

## <a name="roslyn-extensibility"></a>Roslyn genişletilebilirliği
 Genişletilebilirlik: [.net Compiler platform ("Roslyn") genişletilebilirliği](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md)Için Roslyn 'yi nasıl kullanacağınızı öğrenin.

## <a name="azure-sdk"></a>Azure SDK
 Azure SDK 'sını etkinleştirmeyi öğrenin: [Azure SDK 'Sını etkinleştirme](../extensibility/enabling-the-azure-sdk.md).
