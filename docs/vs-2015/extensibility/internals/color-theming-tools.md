# <a name="color-theming-tools"></a>Renk Teması Oluşturma Araçları
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

VS SDK, renk temaları oluşturmanıza ve yönetmenize yardımcı olmak için çeşitli araçlar sağlar.  
  
- [VSIX renk düzenleyici](../../extensibility/internals/vsix-color-editor.md) Aracı, Visual Studio için özel renkler oluşturabilir ve düzenleyebilir.  
  
- [VSIX Color derleyici](../../extensibility/internals/vsix-color-compiler.md) Aracı, var olan Visual Studio Theme. xml dosyalarını uzantınızın. pkgdef biçimine dönüştürür.
