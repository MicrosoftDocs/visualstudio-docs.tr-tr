# <a name="c6216"></a>C6216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C6216: anlamsal olarak farklı tamsayı türleri arasında derleyicinin eklediği dönüştürme: HRESULT için Boole türü  
  
 Bu uyarı, bir Boole 'nin `HRESULT` açıkça yayınlanmadan olarak kullanıldığını gösterir. Boole türleri sıfır olmayan bir değere göre başarıyı gösterir; içindeki Success ( `S_OK` ) `HRESULT` 0 değeri ile belirtilir. Boolean false döndüren işlevlerin tipik hata değeri, bir olarak test edildiğinde başarı durumudur `HRESULT` . Bunun nedeni yanlış sonuçlara yol açabilir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod bu uyarıyı oluşturur:  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  // code ...  
  return IsEqual(riid1, riid2);    
}  
```  
  
 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  if (IsEqual(riid1, riid2) == TRUE)  
  {  
    // code ...  
    return S_OK;  
  }  
  else  
  {  
    // code ...  
    return E_FAIL;  
  }  
}  
```  
  
 Bu uyarı için `SCODE` tür ile eşdeğerdir `HRESULT` .
