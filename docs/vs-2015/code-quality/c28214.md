# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28214: öznitelik parametre adları P1 olmalıdır... P9 olmalıdır  
  
 Bu uyarı, bir ek açıklama bildirimi oluşturduğunuzda parametre adlarının P1 ile sınırlı olduğunu gösterir... P9 olmalıdır. Bu uyarı, normal kullanımda asla gerçekleşmemelidir.
