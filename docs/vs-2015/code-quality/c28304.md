# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28304: C++ başvuru parametresi < parameter_name >, yanlış yerleştirilmiş bir `_Notref_` uygulanan işleci bulundu \<token >.  
  
 `_Notref_` İşleci, yalnızca C++ başvuru parametreleri içeren özel durumlar ve yalnızca sistem tarafından sağlanan makroları kullanılmalıdır. Tarafından hemen gelmelidir bir `_Deref_` işleç veya işlev bir ek açıklama.
