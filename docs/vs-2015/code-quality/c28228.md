# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28228: ek açıklama parametresi: ek açıklamalarda tür kullanılamaz  
  
 Bu uyarı, bir parametrenin desteklenmeyen türde olduğunu gösterir. Ek açıklamalar yalnızca parametre olarak sınırlı bir tür kümesi kullanabilir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
