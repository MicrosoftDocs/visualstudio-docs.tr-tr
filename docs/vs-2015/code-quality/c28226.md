# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28226: Ek açıklama da aynı zamanda (geçerli bildirim içinde) PrimOp olamaz  
  
 Bu uyarı, adlı ek açıklama aynı zamanda PrimOp bildirilmiş ve aynı zamanda normal bir ek açıklama olarak önceden bildirildi gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
