# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28021: açıklama eklenen parametrenin \<param> \<anno> bir işaretçi olması gerekir  
  
 Bu uyarı, açıklama eklenen nesne bir işaretçi türü olmadığında bildirilir. Bu ek açıklama `void` veya integral türleriyle birlikte kullanılamaz.
