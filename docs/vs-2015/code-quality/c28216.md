# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28216: \_ Check_return \_ ek açıklaması yalnızca belirli işlev parametresi için son koşullar için geçerlidir.  
  
 `_Check_return_`Ek açıklama dışında bir bağlamda uygulandı `__post` ; bir `__post` (veya `__drv_out` ) değiştiriciye ihtiyacı olabilir veya hatalı yerleştirilmiş olabilir.
