# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28351: ek açıklama, \<annotation> koşulda bir dinamik değerin (bir değişken) kullanılamayacağını açıklar.  
  
 Bu uyarı, C/C++ türünün incelenebildiği bir ek açıklama uygulandığında raporlanır.  
  
 İçindeki ifadesi `_When_` bir sabit olarak değerlendirilmelidir. Yok `_When_` sayılır.
