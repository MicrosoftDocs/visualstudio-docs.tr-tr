# <a name="c6279"></a>C6279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C6279: \<variable> skalar New ile ayrılmış, dizi delete [] ile silindi  
  
 Bu uyarı yalnızca C++ kodunda görünür ve çağıran işlevin, skalar **New** işleciyle, ancak array **delete []** işleci ile serbest bırakılmış belleğe sahip olduğunu gösterir. Bellek skaler **Yeni**ile ayrılmışsa, genellikle skaler **silme**ile serbest bırakılmalıdır.  
  
 Bunun sorunlara neden olması olası en az üç neden vardır:  
  
- Yok ediciler olsa da, dizideki tek tek nesneler için oluşturucular çağrılmaz.  
  
- Global (veya sınıfa özel) **işleci New** ve **operator delete** işleci **New []** ve **operator delete []** ile uyumlu değilse, beklenmeyen sonuçlar oluşabilir.  
  
  Bu hatanın kesin sonuçları tahmin etmek zordur. Oluşturucular çağrılmadığı için başlatılmamış belleğin kullanımı nedeniyle rastgele davranışa veya kilitlenmelere neden olabilir. Ya da, işleçlerin geçersiz kılındığı durumlarda bellek ayırmaya ve kilitlenmelere neden olabilir. Nadir durumlarda, uyuşmazlık önemli olmayabilir. Analiz Aracı şu anda bu durumlar arasında ayrım yapmaz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod bu uyarıyı oluşturur:  
  
```cpp  
class A  
{  
  // members  
};  
  
void f ( )  
{  
  A *pA = new A;  
  //code ...  
  delete[] pA;  
}  
```  
  
 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:  
  
```cpp  
void f( )  
{  
  A *pA = new A;  
  //code ...  
  delete pA;  
}  
```  
  
 Bu tür ayırma sorunlarını tamamen önlemek için, C++ Standart Şablon kitaplığı (STL) tarafından sunulan mekanizmaları kullanın. Bunlar arasında [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)ve [vektör](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f)bulunur. Daha fazla bilgi için bkz. [akıllı işaretçiler](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) ve [C++ standart kitaplığı](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
