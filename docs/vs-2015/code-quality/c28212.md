# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28212: ek açıklaması için beklenen işaretçi ifadesi  
  
 Bu uyarı, bir ek açıklama (Not eklenen işlev değil) numaralı parametre bir işaretçi ya da dizi türü olması bekleniyor, ancak başka bir türü tespit edildi gösterir. Ek açıklama düzeltilmesi gerekiyor.
