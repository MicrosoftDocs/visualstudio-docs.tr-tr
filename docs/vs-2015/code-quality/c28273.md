# <a name="c28273"></a>C28273
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28273: işlev Için ipuçları işlev bildirimiyle tutarsız  
  
 Bu uyarı, çözümlenmekte olan kodda değil, ek açıklamalarda bir hata olduğunu gösterir. Bir işlev tanımında görüntülenen ek açıklamalar, bildirimde görünenler ile tutarsız. İki ek açıklama eşleşecek şekilde çözümlenmelidir.
