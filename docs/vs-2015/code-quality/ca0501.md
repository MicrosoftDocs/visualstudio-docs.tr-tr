# <a name="ca0501"></a>CA0501

Çıkış raporu okunamıyor.

Bu durum, aşağıdaki nedenlerden biri nedeniyle çıkış rapor yazılamaz oluşur:

-   Disk dolu

-   Dizine yazma izni yok

-   Dizin mevcut değil
