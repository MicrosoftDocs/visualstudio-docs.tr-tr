# <a name="ca0503"></a>CA0503

Özelliği kullanım dışıdır. Yerine geçen özelliği kullanın.

Kullanım dışı bir kod analizi özelliği derleme veya proje yapılandırma dosyasında bir değer içerdiğinde, uyarı CA0503 oluşturulur. Bu uyarıyı çözmek için, onun yerine geçen belirtilen özelliği kullanın.
