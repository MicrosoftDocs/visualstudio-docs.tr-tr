# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28232: \_ \_ \_ \_ \_ \_ herhangi bir ek açıklamaya ön, post veya deref uygulanmadı  
  
 Bu uyarı `_Pre_` , bir, `_Post_` veya `_Deref_` işlecinin sonraki işlevsel ek açıklama olmadan bir ek açıklama ifadesinde göründüğünü gösterir; değiştirici yoksayıldı, ancak bu, yanlış kodlanmış bir ek açıklama gösterir.
