# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28287: işlev için söz dizimi hatası, \_adresindeki\_() ek açıklama (tanınmayan parametre adı)  
  
 Kod Analizi aracına olduğunda bu uyarı raporları `SAL_at` (`__drv_at`) geçerli bağlamda parametresi ifade yorumlanamıyor ve ek açıklama kullanılır. Bu, yanlış yazılmış bir parametre veya üye adı veya bir yazım hatası "return" veya "Bu" anahtar sözcüklerini kullanarak içerebilir.
