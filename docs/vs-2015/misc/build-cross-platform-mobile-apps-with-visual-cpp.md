# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>Visual C++ platformlar arası mobil uygulamalar oluşturun
Çoklu Platform Mobil Uygulama Geliştirme için Visual C++ kullanarak Android, iOS ve Windows cihazları için platformlar arası kod oluşturabilirsiniz. Bu, Visual Studio 2015 ' de sunulan ve iOS, Android ve Windows için kod Visual C++ kullanarak platformlar arası geliştirmeyi sağlayan isteğe bağlı bir özelliktir.  
  
 Yükleme yönergeleri ve kod örnekleri için bkz. [Çoklu platform mobil uygulama geliştirme için Visual C++](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md).
