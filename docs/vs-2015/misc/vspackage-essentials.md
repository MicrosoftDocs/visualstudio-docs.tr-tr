# <a name="vspackage-essentials"></a>VSPackage temelleri
VSPackages, UI öğeleri, hizmetler, projeler, düzenleyiciler ve tasarımcılarla Visual Studio 'Yu oluşturan ve genişleten yazılım modüllerdir. VSPackages, merkezi mimari birimidir [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] . [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] genellikle bir VSPackages koleksiyonu olarak yazılır.  
  
 Bir [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] genişletilebilirlik uygulaması oluşturmak için önce bir VSPackage oluşturun. Daha sonra IDE 'yi genişletmek için menü komutları, araç pencereleri, belge pencereleri veya başka bir kod ekleyebilirsiniz [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] .  
  
 VSPackages 'yi kullanmak için Visual Studio SDK 'sını yüklemeniz gerekir. Bu yönergeyi izlemek için, Visual Studio SDK 'sını yüklemelisiniz. Daha fazla bilgi için bkz. [Visual STUDIO SDK](../extensibility/visual-studio-sdk.md).
