# <a name="the-designer-cannot-be-modified-while-debugging"></a>Tasarımcı hata ayıklama sırasında değiştirilemez
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

  
Uygulama hata ayıklama modunda çalışırken O/R Tasarımcısı öğeleri değiştirmek için bir girişimde, bu ileti görünür. Uygulama hata ayıklama modunda çalışırken, O/R Tasarımcısı salt okunur.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tıklayın **hata ayıklamayı Durdur** üzerinde **hata ayıklama** menüsü.  
  
     Uygulama hata ayıklamasını durdurur ve O/R Tasarımcısı'nda öğeler değiştirilebilir.
