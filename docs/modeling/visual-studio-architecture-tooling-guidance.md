# <a name="visual-studio-architecture-tooling-guidance"></a>Visual Studio Mimari Araç Kullanımı Kılavuzu

Bu Mimari Araç Kılavuzu, Visual Studio 2010'da modelleme araçlarının kullanımı hakkında indirilebilir bir pratik rehberlik paketidir. Tüm modelleme araçlarının tümleşik bir şekilde nasıl kullanıla ilgili üst düzey bir görünüm sunar ve araçların kullanıldıklarına en yaygın senaryoları ele almaktadır.

İçerik, Visual Studio Ürün Ekibi, Microsoft Hizmetleri, Microsoft En Değerli Profesyoneller (MVP'ler) ve Visual Studio Community Müşteri Adayları'nın üyelerine sahip olan Visual Studio alm Rangers tarafından oluşturulmuş.
