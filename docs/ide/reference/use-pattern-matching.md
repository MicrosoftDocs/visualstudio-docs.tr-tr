# <a name="use-pattern-matching"></a>Model eşleştirmeyi kullan

Bu yeniden düzenleme için geçerlidir:

- C#

**Ne:** Model eşleştirmeyi kullanın.

**Ne zaman:** Yeni [C# deseninin eşleşen](https://docs.microsoft.com/dotnet/csharp/whats-new/csharp-9#pattern-matching-enhancements) yeteneklerini kullanmak istiyorsunuz.

**Neden:** Bu yeniden düzenleme, kodunuzu daha net ve kısa hale getirir.

## <a name="how-to"></a>Nasıl yapılır

1. Giriş işaretini deyimin içine yerleştirin.

2. **CTRL** tuşuna basın + **.** **hızlı eylemleri ve yeniden düzenlemeler** menüsünü tetiklemek için.

3. **Desen eşleştirme kullan** 'ı seçin.
