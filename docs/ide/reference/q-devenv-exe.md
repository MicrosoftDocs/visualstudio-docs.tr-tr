# <a name="-devenvexe"></a>/? (devenv.exe)

`devenv`Her anahtarın kısa bir açıklamasıyla birlikte tüm anahtarları listeleyerek bir ileti kutusu görüntüler.

## <a name="syntax"></a>Syntax

```shell
devenv /?
```
