# <a name="ca0062"></a>CA0062

Kural '*RuleId*'Kural kümesinde başvurulan'*RuleSetName*' bulunamadı.

Bu hata, belirtilen kural bulunamadı gösterir.

Bu hata, genellikle bir kural kümesi dosyasını elle düzenlendiği kaynaklanır. Bu hatayı önlemek için Visual Studio'yu kullanabilirsiniz [kural kümesi düzenleyici](../code-quality/working-in-the-code-analysis-rule-set-editor.md) kod çözümleme kurallarını yapılandırmak için.

Bu sorunu çözmek için emin olun tüm onay belirtilen kimlikleri, *.ruleset* dosya geçerlidir. Standart bir kural kümesi dosyası kullanıyorsanız, ilgili kural ipucu yolları dosyasında belirtilir emin olun.
