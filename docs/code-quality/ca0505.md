# <a name="ca0505"></a>CA0505

Geçersiz kılınan Özellik tanımlandığından, kullanım dışı bırakılan özellik yok sayılacak.

Uyarı CA505, hem kullanım dışı hem de yerine geçen kod analizi özellikleri bir proje ya da yapı yapılandırma dosyasında değerler olduğunda oluşturulur. Bu sorunu çözmek için kullanımdan kaldırılan özelliği kaldırın.
