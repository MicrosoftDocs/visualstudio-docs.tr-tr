# <a name="ca0505"></a>CA0505

Superceding özelliği tanımlanmadığından kullanım dışı bırakılan özellik gözardı edilecek.

Uyarı CA505 hem de kullanım dışı ve önceki uygulamaların yerini alan kod analizi özelliklerini değerlere sahip bir proje veya derleme yapılandırma dosyası oluşturulur. Bu sorunu çözmek için kullanım dışı özelliği kaldırın.
