# <a name="ca0055"></a>CA0055
Dosya yüklenemedi:\<path>

Kod Analizi Aracı, analiz için belirtilen hedef dosyayı yükleyemedi. Dosya bulunamazsa veya dosya erişimi reddedildiyse bu durum oluşur. Genellikle bu durum, Kullanıcı Visual Studio 'da giriş derleme özelliğini geçersiz kılmayı seçerse veya fxcopcmd.exe el ile çağırırdı.
