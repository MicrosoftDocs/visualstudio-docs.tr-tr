# <a name="ca0502"></a>CA0502
Desteklenmeyen dil

Bu hata, Kullanıcı bir veya daha fazla sayfa ya da Visual Basic C# dışındaki bir dilde yazılmış bir veya daha fazla sınıf içeren bir ASP.NET projesi çözümlemeyi denediğinde oluşur. Bu hatayı onarmak için, dosyayı projeden kaldırın veya Kod analizini devre dışı bırakın.
