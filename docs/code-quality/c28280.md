# <a name="c28280"></a>C28280
C28280 Uyarı: simgesi için bir 'end' bir eşleşen olmadan 'başlamak' bulundu

 Başlangıç ve bitiş tarihi ek açıklama dili destekler (`{` ve `}` c) yapı ve eşleştirme onayınızı dengesiz. Makrolar kullanılıyorsa, bu durum önlenebilir.
