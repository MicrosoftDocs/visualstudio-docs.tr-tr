# <a name="c28243"></a>C28243
C28243 Uyarı: işlevi parametresi üzerinde daha gerektirir açıklama gerçek tür izin verdiğinden ek açıklamanın dereferences

 Sayısı `__deref` açıklamanın işleçlerini olan birden çok parametre türü tarafından tanımlanan işaretçinin düzey sayısı. Bu uyarı ya da değiştirerek ek açıklama veya başvurulan parametre işaretçi düzeylerini sayısında düzeltin.
