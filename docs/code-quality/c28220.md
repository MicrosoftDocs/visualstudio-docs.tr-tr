# <a name="c28220"></a>C28220
C28220 Uyarı: tamsayı ifade ek açıklamanın parametresinde eklenti için bekleniyor

 Bu uyarı, açıklamanın parametresi bir tamsayı ifade olması beklenir ve bazı diğer türüyle karşılaşıldı gösterir. Bu genellikle bir yanlış kodlanmış ek açıklama makrosu gösterir.
