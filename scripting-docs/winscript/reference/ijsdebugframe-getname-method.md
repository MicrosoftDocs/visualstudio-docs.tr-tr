# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName Metodu
Yığın çerçevesi kolay kullanılan adını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pName`  
 [out] Yığın çerçevesi adı.  
  
## <a name="return-value"></a>Dönüş Değeri  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jscript9diag.h
