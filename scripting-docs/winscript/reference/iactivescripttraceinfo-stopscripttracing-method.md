# <a name="iactivescripttraceinfostopscripttracing-method"></a>IActiveScriptTraceInfo::StopScriptTracing Yöntemi
Komut dosyası izlemeyi durdurur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT StopScriptTracing(void); }  
```  
  
#### <a name="parameters"></a>Parametreler  
 Bu yöntem hiç parametre yok.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Bu yöntemin dönüş olası değerler şu şekildedir:  
  
1.  S_OK: başarılı.  
  
2.  E_NOTIMPL: uygulanmadı.
