# <a name="idebugsessionprovider-interface"></a>IDebugSessionProvider Arabirimi
Hata ayıklayıcı IDE ana bilgisayar ve dil etkinleştirmek için sağlanan birincil arabirimi hata ayıklama başlattı. Çalışan bir uygulama için bir hata ayıklama oturumu oluşturur. Bu arabirim Makine Hata Ayıklama Yöneticisi tarafından uygulanır.  
  
 Kaynağından devralındı yöntemleri yanı sıra `IUnknown`, `IDebugSessionProvider` arabirimi aşağıdaki yöntemleri sunar.  
  
## <a name="methods-in-vtable-order"></a>Vtable sırayla yöntemleri  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IDebugSessionProvider::StartDebugSession](../../winscript/reference/idebugsessionprovider-startdebugsession.md)|Belirtilen uygulama ile bir hata ayıklama oturumu başlatır.|
