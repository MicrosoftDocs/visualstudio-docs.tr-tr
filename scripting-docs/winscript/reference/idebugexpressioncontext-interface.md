# <a name="idebugexpressioncontext-interface"></a>IDebugExpressionContext Arabirimi
Burada ifadeleri değerlendirilebilir bağlamını temsil eder. Yığın çerçeve nesnesi bu arabirimi uygular.  
  
 Kaynağından devralındı yöntemleri yanı sıra `IUnknown`, `IDebugExpressionContext` arabirimi aşağıdaki yöntemleri sunar.  
  
## <a name="methods-in-vtable-order"></a>Vtable sırayla yöntemleri  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IDebugExpressionContext::ParseLanguageText](../../winscript/reference/idebugexpressioncontext-parselanguagetext.md)|Belirtilen metni için bir hata ayıklama ifadesi oluşturur.|  
|[IDebugExpressionContext::GetLanguageInfo](../../winscript/reference/idebugexpressioncontext-getlanguageinfo.md)|Bu bağlamın sahip dil için bir ad ve GUID döndürür.|
