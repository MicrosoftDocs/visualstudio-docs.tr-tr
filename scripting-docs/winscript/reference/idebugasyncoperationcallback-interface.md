# <a name="idebugasyncoperationcallback-interface"></a>IDebugAsyncOperationCallBack Arabirimi
İlerleme durumunu ilgili durum olaylarını sağlayan bir `IDebugAsyncOperation` arabirim değerlendirme.  
  
## <a name="methods"></a>Yöntemler  
 Kaynağından devralındı yöntemleri yanı sıra `IUnknown`, `IDebugAsyncOperationCallBack` arabirimi aşağıdaki yöntemleri sunar.  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Sonuçta bir zaman uyumsuz hata ayıklama işlemi kullanılabilir olduğunu bildirir.|
