' Var ' bekleniyor (ERRnoVar 1034))  
  
 Gerektiğinde, anahtar sözcüğünü ekleme işlemi başarısız oldu.  
  
 Var anahtar sözcüğü hakkında daha fazla bilgi için bkz. [var bildirisi](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Statements/var).
