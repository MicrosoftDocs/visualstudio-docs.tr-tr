# <a name="default-can-only-appear-once-in-a-switch-statement"></a>'default', 'switch' deyimi içinde yalnızca bir kere kullanılabilir.
Bir switch ifadesinin içinde **Default** ifadesini birden çok kez kullanmaya çalıştınız. Varsayılan durum, Switch deyimindeki her zaman son Case deyimidir (Bu durum, karşı bir durumdur).  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyiminizden tüm ek **varsayılan** Case deyimlerini kaldırın `switch` (Switch deyiminizde en çok bir varsayılan Case deyimi kullanın).
