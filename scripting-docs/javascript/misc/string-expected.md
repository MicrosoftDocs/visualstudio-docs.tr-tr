# <a name="string-expected"></a>Dize bekleniyor
Çağrılacak çalıştı **String.prototype.toString** veya **String.prototype.valueOf** dışında bir türde bir nesne üzerinde yöntemi `String`. Bu tür çağırma nesne türünde olmalıdır `String`.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yalnızca çağırma **String.prototype.toString** veya **String.prototype.valueOf** türündeki nesneleri yöntemlere `String`.
