# <a name="string-expected"></a>Dize bekleniyor
Dışındaki bir türün nesnesi üzerinde **String. prototype. ToString** veya **String. prototype.** bir yöntemi çağırmayı denediniz `String` . Bu tür çağrının nesnesinin türünde olması gerekir `String` .  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yalnızca türündeki nesnelerde **String. prototype. ToString** veya **String. prototype.** bir yöntemi çağırın `String` .
