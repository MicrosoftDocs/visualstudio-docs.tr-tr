# <a name="expected-end"></a>' \@ End ' bekleniyor
Koşullu koda derlenmiş bir kod bloğu oluşturmaya çalıştınız, ancak **@end** sonunda ifadesini içermiyordu. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] deyimler bir blok içinde çevreleyerek koşullu olarak derlenebilir <strong>@if/@end</strong> .  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Karşılık gelen <strong>@end</strong> ifadesini ekleyin.
