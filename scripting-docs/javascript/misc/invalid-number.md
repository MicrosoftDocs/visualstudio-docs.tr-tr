# <a name="invalid-number"></a>Geçersiz sayı
Geçersiz sayı (ErrbadNumber 1013)  
  
 JavaScript içinde geçerli olmayan bir sayı kullanmaya çalıştınız.  
  
 Geçerli sayılar hakkında daha fazla bilgi için bkz. [Number Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number).
