# <a name="regular-expression-object-expected"></a>Normal ifade nesnesi bekleniyor
Dışındaki bir türün nesnesi üzerinde **RegExp. prototype. ToString** veya **RegExp. prototype.** bir yöntemi çağırmayı denediniz `RegExp` . Bu tür çağrının nesnesinin türünde olması gerekir `RegExp` .  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yalnızca türündeki nesnelerde **RegExp. prototype. ToString** veya **RegExp. prototype.** bir yöntemi çağırın `RegExp` .
