# <a name="the-precision-is-out-of-range"></a>Duyarlılık aralık dışı
İşleve geçersiz bir bağımsız değişken geçirme girişiminde **Number.prototype.toPrecision**. Bağımsız değişkeni **toPrecision** 1-21 (dahil) arasında olmalıdır.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişkeni olun `toPrecision` çok büyük veya çok küçük değil.
