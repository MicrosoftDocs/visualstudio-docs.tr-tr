## <a name="how-do-my-developers-activate-software-using-product-keys-from-their-visual-studio-subscription"></a>Geliştiricilerim Visual Studio aboneliklerinin ürün anahtarlarını kullanarak nasıl yazılım etkinleştirebilir?

Abonenizin hangi ürünü indirdiğine bağlı olarak, ürün aboneliğe bağlı e-posta adresiyle oturum açarak etkinleştirilebilir veya aboneler Visual Studio Abone portalından [bir ürün anahtarını kullanarak](../../../../product-keys.md) ürünü etkinleştirebilir.
