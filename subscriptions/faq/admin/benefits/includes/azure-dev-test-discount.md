## <a name="how-do-my-developers-get-50-off-azure-devtest-workloads"></a>Geliştiricilerim Azure Geliştirme/Test iş yüklerinde nasıl %50 indirim alabilir?

Kuruluşunuz Azure için Kurumsal Anlaşması varsa geliştiricileriniz 50 ABD Doları-150 ABD Doları Azure kredisi ile Geliştirme/Test fiyatında %50 indirim arasında seçim yapabilir. Aynı abonelikte ikisini birden kullanamaz. Geliştiricilerinize Geliştirme/Test fiyatlandırmasını etkinleştirme işleminin ayrıntıları için, [Abonelerinizin Azure’dan en iyi şekilde yararlanmasına yardımcı olma](https://aka.ms/HelpingSubscriberswithAzure) videosunu gözden geçirin.
