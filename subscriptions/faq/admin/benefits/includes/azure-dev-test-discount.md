## <a name="how-do-my-developers-get-50-off-azure-devtest-workloads"></a>Geliştiricilerim Azure geliştirme ve test iş yüklerinde nasıl %50 indirim elde ediyor?

Kuruluşta Azure için Kurumsal Anlaşma varsa, aboneler çeşitli Azure hizmetleri için indirimli Azure geliştirme/test fiyatlarına sahip olabilir ve bu da uygun maliyetli geliştirme ve test olanaklarını sağlar. Geliştiricilerinize Geliştirme/Test fiyatlandırmasını etkinleştirme işleminin ayrıntıları için, [Abonelerinizin Azure’dan en iyi şekilde yararlanmasına yardımcı olma](https://aka.ms/HelpingSubscriberswithAzure) videosunu gözden geçirin.
